== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Ajax Comment With Sweet Alert'
3. Activate Ajax Comment With Sweet Alert from your Plugins page.

= From WordPress.org =

1. Download Ajax Comment With Sweet Alert.
2. Upload the 'Ajax Comment With Sweet Alert' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Ajax Comment With Sweet Alert from your Plugins page.
